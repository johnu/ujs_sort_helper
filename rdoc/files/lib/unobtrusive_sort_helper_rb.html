Helpers to sort tables using clickable column headers.

Author: Stuart Rackham <srackham@methods.co.nz>, March 2005. Modified by: Wynn Netherland <wynn@squeejee.com>, February 2008 to use UJS License: This source code is released under the MIT license.

  • Consecutive clicks toggle the column‘s sort order.
  • Sort state is maintained by a session hash entry.
  • Icon image identifies sort column and state.
  • Typically used in conjunction with the Pagination module.

Example code snippets:

Controller: 

  helper :sort
  include SortHelper

  def list
    sort_init 'last_name'
    sort_update
    @items = Contact.find_all nil, sort_clause
  end

Controller (using Pagination module): 

  helper :sort
  include SortHelper

  def list
    sort_init 'last_name'
    sort_update
    @contact_pages, @items = paginate :contacts,
      :order_by => sort_clause,
      :per_page => 10
  end

View (table header in list.rhtml): 

  <thead>
    <tr>
      <%= sort_header_tag('id', :title => 'Sort by contact ID') %>
      <%= sort_header_tag('last_name', :caption => 'Name') %>
      <%= sort_header_tag('phone') %>
      <%= sort_header_tag('address', :width => 200) %>
    </tr>
  </thead>
  • The ascending and descending sort icon images are sort_asc.png and sort_desc.png and reside in the application‘s images directory.
  • Introduces instance variables: @sort_name, @sort_default.
  • Introduces params :sort_key and :sort_order.
